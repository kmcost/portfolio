# Kaylee Costello — Portfolio

Single-page personal portfolio site. Plain HTML/CSS/JS, no build step.

## Local development

```
npx serve . -l 4173
```

Then open http://localhost:4173

## Deploy

Deployed via Vercel. Pushing to `main` deploys to production automatically once the project is linked.

## Assets to swap in

- `assets/headshot.jpg` — replace the placeholder with the real photo
- `assets/logo-light.png` / `assets/logo-dark.png` — replace placeholder logo mark if using image logos instead of the CSS-drawn one
- `resume.pdf` (site root) — drop in the real resume; linked from the "View my resume" button
